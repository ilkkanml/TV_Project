# Nexora TV — Studio Constitution

## Core Authority Structure

Nexora TV development is directed by studio leadership.

All departments operate under this authority structure.

No department may independently:

- add systems
- remove systems
- rewrite architecture
- change UX direction
- modify monetization philosophy
- alter backend structure
- replace playback stack
- change visual identity
- change subscription flow
- change platform priorities

without explicit approval from leadership.

---

# Mandatory Approval Rule

Before any implementation change is accepted:

1. Proposal must be reviewed.
2. Direction must be approved.
3. Architecture compatibility must be verified.
4. Product identity must remain protected.

No autonomous feature drift is allowed.

---

# Department Discipline

All departments must:

- follow locked product direction
- preserve premium cinematic identity
- preserve TV-first UX
- preserve hidden API architecture
- preserve playback continuity philosophy
- avoid unnecessary complexity
- avoid random experimentation in production code

Departments execute.
Leadership approves direction.

---

# Architecture Protection Rules

The following are protected systems:

- Auth architecture
- Hidden backend API
- Playback core
- TV navigation behavior
- Compose TV design system
- Session structure
- Device activation flow
- Premium cinematic UI direction

These systems cannot be replaced or rewritten without leadership approval.

---

# Studio Workflow

Departments may:

- propose improvements
- identify risks
- prepare implementation plans
- optimize approved systems
- improve performance
- improve stability

Departments may not:

- silently alter product vision
- inject unrelated systems
- create scope drift
- replace approved technologies
- change roadmap direction
- bypass approval structure

---

# Current Locked Direction

Current approved implementation direction:

- Kotlin
- Android TV
- Compose TV
- ExoPlayer / Media3
- Hidden backend architecture
- Premium cinematic UI
- Remote-first UX
- Netflix-like TV navigation

---

# Final Rule

Nexora TV must evolve as a controlled premium platform.

Not as a chaotic feature collection.
