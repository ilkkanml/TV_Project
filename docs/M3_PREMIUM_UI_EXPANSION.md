# M3 Premium UI Expansion

## Status

M3 is the active milestone after M2 Playback Freeze Lock.

## Goal

Build the premium cinematic TV interface layer for Nexora TV.

## Premium UI Direction

- Futuristic cinematic design
- Dark metallic visual language
- Deep neon cyan accent
- Electric blue secondary glow
- Large cinematic posters
- Dynamic blurred artwork background
- Scale plus glow focus behavior
- Smooth TV-safe transitions
- Netflix-like familiarity without copying Netflix directly

## Home Experience

Locked structure:

1. Continue Watching
2. Featured
3. Live TV
4. Movies
5. Series
6. Favorites

## Hero System

- Auto-sliding featured content
- 8 second timing
- Auto pause on focus or user interaction
- Dynamic artwork background sync
- Premium fade transitions

## Poster System

- Large cinematic posters
- Minimal title overlay
- Focus scale plus glow
- TV-distance readability
- Performance-aware image loading

## Player Visual Rules

- Minimal overlay
- 5 second overlay timeout
- Cinematic fade out
- No clutter
- No mobile-style player controls

## Live TV UI

- Netflix-like poster rows
- Instant fullscreen on channel selection
- Fast channel zap priority
- Smooth return to browsing

## Movies and Series UI

- Detail page before playback
- Muted autoplay preview
- Cinematic artwork background
- Smooth fade into playback

## Out of Scope

- UI overhaul beyond M3 scope
- Heavy sci-fi intro
- Complex recommendation engine
- Excessive motion effects
- Cluttered content pages

## Pass Criteria

M3 is successful when Nexora TV has a consistent premium cinematic TV UI language that can be implemented safely without hurting playback performance.
