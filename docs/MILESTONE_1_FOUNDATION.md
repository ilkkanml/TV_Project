# Milestone 1 — Android TV Foundation

## Goal

Build the first usable Android TV / Fire TV IPTV player foundation.

## Pass Criteria

- App opens on Android TV
- Remote navigation works
- IPTV setup screen exists
- Live TV structure exists
- Player screen structure exists
- Project has a clean modular base

## Initial Screens

- Splash
- Login
- IPTV Setup
- Home
- Live TV
- Player
- Settings

## Out of Scope

- Samsung TV
- LG TV
- Apple TV
- Recording
- Advanced recommendations
- Heavy animations
