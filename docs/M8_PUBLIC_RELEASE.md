# M8 Public Release

## Goal

Prepare Nexora TV for public release.

## Public Release Requirements

- Stable Android TV build
- Stable Fire TV build
- Working auth flow
- Secure hidden backend API
- Subscription validation
- Playback stability
- Premium cinematic UI consistency
- Recovery-safe GitHub milestone backups

## Release Direction

Nexora TV should launch as a premium cinematic IPTV platform with a TV-first experience.
