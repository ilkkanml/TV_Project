# M10 — Ecosystem Alignment & Client Integration Contract

This document defines the core ecosystem contract for Nexora TV and Android client integration. It records M10 opening changes.

- TV_Project = Android TV / Fire TV first client
- TV_Project_Platform = Core Account / Device / License / Admin / Remote Config / App Version / Profile Transfer center
- Ecosystem Integration department added
- First app free until final release
- Payment/subscription enforcement inactive during early launch
- No content hosting, broadcasting, channel selling, bundled streams, unauthorized scraping, DRM bypass, credential sharing
- User-owned / legally authorized sources only
