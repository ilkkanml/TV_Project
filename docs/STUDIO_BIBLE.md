# TV Project — Studio Bible

## Vision

TV Project is a premium smart TV streaming platform focused on simplicity, stability, and familiarity.

The application should feel cinematic, modern, fast, and comfortable for long viewing sessions.

---

# Product Philosophy

## Core Identity

- TV-first
- Remote-first
- Stability-first
- Clean UI
- Premium feel

## Design Direction

The UI should combine:

- Netflix familiarity
- IPTV practicality
- TV readability

---

# Non-Negotiable Rules

1. Playback stability is more important than visual effects.
2. Focus navigation must always remain clear.
3. The UI must remain readable from distance.
4. The app must feel calm and uncluttered.
5. The project must remain a legal player platform.

---

# Technical Direction

## Primary Stack

- Kotlin
- Jetpack Compose for TV
- Media3 ExoPlayer
- Retrofit
- OkHttp
- Supabase
- Stripe

---

# MVP Goal

The first milestone goal:

User enters IPTV credentials → categories load → channel opens.

---

# Current Priority

Build the strongest Android TV foundation possible before platform expansion.
