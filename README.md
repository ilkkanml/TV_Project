# TV Project

A premium Android TV / Fire TV IPTV player foundation.

## Product Direction

TV Project is designed as a legal, subscription-based IPTV player application. The app provides a clean TV experience where users can sign in, manage their subscription, connect their own IPTV account, and watch live TV, movies, and series through supported legal IPTV sources.

## Platform Target

- Android TV
- Fire TV

Future targets may include Samsung Tizen, LG webOS, and Apple tvOS after the Android TV foundation is stable.

## Core Principle

This repository must remain a player application. It must not include pirated channel lists, embedded illegal streams, or bundled unauthorized content.

## v0.1 MVP Goal

The first milestone focuses on a stable foundation:

- TV app shell
- Dark premium UI direction
- IPTV credential entry
- Xtream Codes API connection layer
- Live TV category and channel foundation
- Media3 ExoPlayer playback shell
- Remote-friendly navigation
